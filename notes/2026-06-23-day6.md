## Exp020 Results

Experiment Name

Exp020 — Whisper Large-v3 LoRA attention plus feed-forward layers

Configuration

- Base model: Whisper Large-v3
- LoRA rank: 16
- LoRA alpha: 32
- Target modules:
  - q_proj
  - k_proj
  - v_proj
  - out_proj
  - fc1
  - fc2
- Learning rate: 2e-5
- Max steps: 1000

Results

Best WER: 0.2278

Reference Baseline

- Exp018 WER: 0.2637

Improvement

- Absolute improvement: 0.0359
- Relative improvement: 13.6%

Observations

- Largest improvement observed in the Large-v3 LoRA series.
- Feed-forward adaptation appears highly beneficial.
- Best checkpoint occurred before the end of training.
- Evidence of overfitting after the best checkpoint.
- Expanded attention + feed-forward adapters become the new champion architecture.

Conclusion

New champion configuration:

q_proj, k_proj, v_proj, out_proj, fc1, fc2

Best WER:

0.2278

Recommended next step:

Investigate training duration or LoRA capacity while keeping the new architecture fixed.
