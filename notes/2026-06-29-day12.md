## Experiment 036A — AfriVoices Dataset Profiling

### Objective

Build a reusable dataset profiler to inspect the complete AfriVoices competition corpus before creating a unified training manifest.

### Major Accomplishments

- Successfully authenticated and accessed all gated Hugging Face datasets.
- Implemented a six-language profiler supporting seven source repositories.
- Successfully profiled:
  - Swahili
  - Kikuyu
  - Luo (Dholuo)
  - Somali
  - Maasai
  - Kalenjin
- Generated:
  - afrivoices_profile.md
  - afrivoices_language_summary.csv
  - afrivoices_split_summary.csv
  - afrivoices_top_dialects.csv
  - visualization figures
- Produced a reusable profiling pipeline for future datasets.

### Dataset Statistics

- Total utterances: 1,081,999
- Total measured audio: 5,542.97 hours
- Languages: 6
- Speakers detected: 3,577

Language distribution:

| Language | Hours |
|----------|------:|
| Swahili | 3201.18 |
| Luo | 681.70 |
| Kalenjin | 496.26 |
| Maasai | 482.67 |
| Somali | 478.19 |
| Kikuyu | 202.97 |

### Key Findings

- The corpus is highly imbalanced, with Swahili contributing the majority of training hours.
- Kaggle evaluates languages equally despite the imbalance, making future language-balancing experiments potentially valuable.
- Scripted and unscripted speech distributions differ substantially between languages.
- Swahili speaker metadata appears incomplete in the current parser implementation and should be revisited later if speaker-level analyses become important.
- The profiler successfully validates that all competition datasets are accessible and ingestible.

### Outcome

Experiment 036A successfully completed.

The project is now ready to build unified training manifests in Exp036B before beginning the first full AfriVoices training run.
