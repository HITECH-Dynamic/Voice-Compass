# Exp055 — Results

## Objective
Diagnose why Exp054 true generation WER was poor after switching from teacher-forced proxy WER to generation-based WER.

## Diagnostic Work
Checked the clean evaluation dataset and confirmed:
- 98 evaluation rows
- 0 empty transcripts
- balanced language distribution across kik, kln, luo, mas, som

Investigated the audio index and dataset-builder behavior.

## Key Finding
The active ANV audio index only contained Kikuyu:

- kik: 150,515 rows
- kln: 0 rows
- luo: 0 rows
- mas: 0 rows
- som: 0 rows

This explains why the dataset builder only resolved indexed shards for Kikuyu while the other four languages had `unique_parquet_shards = 0`.

## Conclusion
Exp055 identified the issue as an incomplete audio index, not a Whisper/LoRA/generation problem.

The next step is to rebuild a proper five-language ANV audio index before rerunning generation-based training/evaluation.
