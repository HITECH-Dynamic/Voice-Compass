## Exp025 Results

Experiment Name

Exp025 — Whisper Large-v3 LoRA learning rate 3e-5 study

Configuration

Same as Exp020 except:

- Learning rate: 2e-5 → 3e-5

Results

- Best WER: 0.2257
- Final WER: 0.2371

Reference Baseline

- Exp020 best WER: 0.2278

Conclusion

Exp025 reached the lowest observed WER so far at 0.2257, but did not maintain that performance through the end of training.

The higher learning rate learned faster and briefly outperformed Exp020, but performance degraded after the best checkpoint.

Decision

Do not promote Exp025 as the current champion because the final saved model underperformed Exp020.

However, Exp025 is highly informative and should be revisited with early stopping or scheduler changes.

Current champion remains:

Exp020 — Whisper Large-v3 LoRA qkvo + fc1/fc2

Best stable WER:

0.2278

Important note:

Exp025 produced the best observed checkpoint WER so far:

0.2257

Recommended next direction:

Learning-rate scheduler or best-checkpoint saving strategy.
