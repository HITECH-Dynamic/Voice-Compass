## Findings
The model quality is intentionally poor because this was only a 5-step infrastructure validation run.

The major bottleneck discovered was dataset layout rather than model training:
- Although audio indexing eliminated shard searching,
- Samples were randomly distributed across dozens of 700–770 MB parquet shards,
- Evaluation repeatedly downloaded many large shards,
- Total runtime was approximately 39 minutes despite only 100 total examples.

## Conclusion
Exp043 successfully validates the complete Whisper → LoRA → Evaluation → WER pipeline.

The next optimization should target dataset locality rather than model hyperparameters.

# Exp044 — Results

## Objective
Reduce experiment runtime through shard-aware sampling while keeping model training unchanged.

## Outcome
✅ Dataset successfully concentrated into a single indexed ANV parquet shard.
✅ Train set: 80 examples.
✅ Eval set: 20 examples.
✅ unique_parquet_shards = 1.
✅ Training pipeline completed successfully.

## Metrics
- Train runtime: ~68 minutes
- Eval runtime: ~48 minutes
- Train loss: 4.170
- Eval loss: 4.705
- Eval WER: 1.115

## Findings

The hypothesis was disproven.

Although shard-aware sampling reduced the dataset to a single parquet shard, runtime became slower rather than faster.

This indicates that the dominant bottleneck is repeated loading/decompression of the same parquet shard rather than the number of unique shards accessed.

## Conclusion

Shard-aware sampling alone is insufficient.

The dataset loader must cache loaded parquet shards in memory before further multilingual scaling.


# Exp045 — Prep

## Objective
Implement in-memory ANV parquet shard caching inside the dataset loader.

## Rationale
Exp044 proved that shard-aware sampling alone is insufficient. Even when the dataset used one shard, runtime remained slow, indicating repeated parquet loading/decompression during sample access.

## Planned Change
Cache each loaded ANV parquet shard in memory after first access and reuse it for all subsequent samples from the same shard.
