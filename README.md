# Voice Compass

A practical offline ASR system for African languages.

## Objective

Develop and evaluate multilingual speech recognition systems that can run offline on low-resource hardware.

## Competition

AfriVoices East Africa ASR Hackathon (2026)

## Status

Phase 1: Base model evaluation

## Current Capabilities

- Offline transcription
- CPU inference
- Faster Whisper integration

## Roadmap

- Base model comparison
- Dataset exploration
- Fine-tuning
- Hardware validation
- Kaggle submission
